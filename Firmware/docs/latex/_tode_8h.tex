\mbox{\hyperlink{class_tode}{Tode}} Class. 


\begin{DoxyPre}
 \mbox{\hyperlink{class_tode}{Tode}} class represents the 'Control' Configuration.  Any Device Settings are stored in System.
 All Radio Settings are stored in System except Radio Address.
 It is a '\mbox{\hyperlink{class_menu_list}{MenuList}}' of Devices.\end{DoxyPre}



\begin{DoxyPre} System Settings will be saved at the tail of EEPROM space.
 ThisTode will be the first \mbox{\hyperlink{class_tode}{Tode}} at the start of EEPROM space.
 Subsequent Remote \mbox{\hyperlink{class_tode}{Tode}} configuration will be stored after ThisTode from start.
 We will no longer use Tode-\/UID but instead use RF-\/Addr + 15 Arduino ID from Peer.
 Pinging will occur on FFFF but be filtered by the 1-\/nibble Arduino ID.
\end{DoxyPre}
