Detailed Description

Represents a Tode Device.

  • EEPROM Organization ( see EE_ in Constants.h )

    • byte[0] = Tode Size in bytes
    • byte[1] = Version
    • byte[2] = UID
    • byte[3~12] = Tode Name 9-Chars + NULL
    • byte[14] = 1st Device Size
