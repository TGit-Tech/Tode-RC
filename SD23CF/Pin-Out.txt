1.8 TFT
----------------------------------------
SCK = (3)ICSP, SCK,  D52
SDA = (4)ICSP, MOSI, D51
RES = (5)ICSP, RESET
RS	= D28
CS	= D29
BL	= D33 (through R3)
----------------------------------------
Digital Keypad
----------------------------------------
SET	= (1)ICSP, (PCINT3), MISO, D50, #22
OK	= D20 (INT1), SDA, #44
RT	= D21 (INT0), SCL, #43
UP	= A12 (PCINT20)
DW	= A14 (PCINT22)
LT	= A15 (PCINT23)
----------------------------------------
RF Radio
----------------------------------------
AUX = D24
RXD = D19 (RXD1)
TXD = D18 (TXD1)
M1	= D22
M0	= D26
----------------------------------------
OTHER
----------------------------------------
D44 = PWM
D46 = BUZZER
